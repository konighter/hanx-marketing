Which interaction design do you prefer for optional attributes?

The goal is to cleanly hide unused elements, but make them easy to add.

A

Modal Selection (Add to Main Form)

A button at the bottom: "+ Add Optional Attribute". Clicking it opens a popup with checkboxes for all available optional attributes. Once selected, they get permanently added into the main form as regular input fields.

B

Inline Dropdown Selector

At the bottom of the form, there's a dropdown "Select attribute to add...". When an attribute is chosen, its input field instantly appears below without leaving the page.

C

Dynamic Key-Value Table

A separate "Additional Information" card containing a table layout. You click "Add Row", pick the attribute name from a left-column dropdown, and fill the value on the right column.
